## ⚡ What it does

A full-stack platform to manage housing society operations:

- 📋 Complaint tracking  
- 🧑‍🔧 Worker assignment  
- 💳 Payment management  
- ⚡ Real-time updates across users  

---

## 🧠 How it works
User Action → Firestore → Real-time Listener → Instant UI Update
